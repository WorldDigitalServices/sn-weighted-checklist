# Weighted Checklist for Standard Notes

A priority checklist editor with weighted items (1-5 scale).

## Features

- Weight scale 1-5 (higher = higher priority)
- Auto-sorting by weight
- Heat colors: blue (low) â†’ red (high priority)

## Installation

Add this extension URL in Standard Notes:

```
https://worlddigitalservices.github.io/sn-weighted-checklist/ext.json
```

## Technical

Single static HTML file using ComponentRelay for Standard Notes communication. No build process required.
